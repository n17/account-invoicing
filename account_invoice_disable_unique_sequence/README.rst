===========================================
Disable unique sequence check from invoices
===========================================

Allows multiple invoices to exist with the same sequence number.
Please be aware of the possible consequences of installing this module!

The purpose of this module is to allow having overlapping sequences TEMPORARILY
in special situations where sequencing needs to be adjusted.
